# Project Description

Our plan is to build a minimal Terraria-like fighting game on the 2D PixiJS library.

## Usage

### Building the Project

Using the terminal in the project directory, install the dependencies

```bash
yarn
```

Start the website

```bash
yarn start
```

### Keyboard Interactivity

| Key | Controller |
|-|-|
| W | Up |
| A | Left |
| S | Down |
| D | Right |
| SPC | Jump |
| Mouse1 | Fire/Attack |
